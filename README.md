# Quantum-Key-Distribution
The repository contains the code that won first place in Qiskit Fallfest 2025 IITM. The code uses Qiskit software to create a simulation of BB84 and EE92 protocols with both Ideal and Noisy simulators. I was part of the team |GoToshDino> along with Aditya Goel (ep24b001@smail.iitm.ac.in) and Toshith (ee23b0xx@smail.iitm.ac.in).
